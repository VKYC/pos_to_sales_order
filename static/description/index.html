POS Home Delivery

This module allows you to manage home deliveries easily. Seller can create a Sales Order in a draft state as quotation in running POS Session. While creating a sales order for home delivery, seller can add another delivery address as per customer’s request. Extra charges can be charged for delivery. Seller can add notes with the order for internal reference which can be seen in the odoo backend orders. In this way, this module makes it easy for the POS User to handle the Sales Orders

Features
  • Seller can create Sale Order in POS running Session.
  • Seller can send orders for home delivery to his valuable customers
  • Seller has an option to mention another delivery addresses.
  • Seller can charge extra amount for delivery or anything.
  • Seller can also add internal notes to the order.

  • POS Session View
  • Backend View & Configuration

Sale Order button

Select customer and add product(s) in cart that you wish to deliver then click on Create Sale Order button to create sale order.

Change Shipping Address for Sale Order

By default shipping address will be customer address but seller can add another delivery address as per customer’s request. To add different shipping address, checked 'Ship to Different Address' check-box & fill details for new shipping address.
